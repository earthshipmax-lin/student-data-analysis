# Student Data Analysis

一个使用 **Python + Pandas** 实现的学生成绩数据分析项目。

## 项目介绍

本项目用于对学生成绩数据进行分析，包括：

- 数据读取
- 数据清洗
- 成绩统计
- 平均分计算
- 可视化分析（后续可扩展）

适合用于：

- Python 数据分析练习
- Pandas 入门学习
- 数据处理基础项目
